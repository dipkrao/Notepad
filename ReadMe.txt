# React Native Note Pad App

This is a simple note-taking application built with React Native and TypeScript. The app allows users to create, edit, delete, and search notes. It also supports color coding and local storage for persistence.

## Features

- **Note Listing:** Display a list of notes on the main page. Each note has a title, content, and color.
- **Create Note:** Add a new note with a title, content, and color. New notes appear at the top of the list.
- **Edit Note:** Edit the title and content of each note. Changes are reflected immediately.
- **Delete Note:** Option to delete a note, with confirmation from the user.
- **Responsive Design:** The application is designed to work well on all mobile devices.
- **Color Coding:** Users can assign different colors to notes for organization.
- **Local Storage:** Notes are saved to local storage to persist even when the app is refreshed or closed.
- **Search Functionality (Bonus):** Filter notes based on their titles or content.
